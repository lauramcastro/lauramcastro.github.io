El sistema desarrollado en este trabajo de fin de grado propone una aplicación web donde se recoge información sobre todos los grados de la UDC (Universidade da Coruña), con el fin de ayudar al alumnado actual (y futuro) a decidir sus matriculaciones anuales en la universidad, mediante la visualización de estadísticas de asignaturas y el intercambio de información con otras personas interesadas. Para cada grado, el sistema muestra las asignaturas que lo componen, y para cada asignatura se muestran estadísticas de los últimos tres años académicos (y una media global de todos los años disponibles), que se componen de: número de matriculados, aprobados, suspensos y no presentados. Esta información puede ser un buen indicador de la dificultad de una asignatura, pero adicionalmente la aplicación web proporciona también un sistema de comentarios por asignatura, en los cuales el alumnado puede intercambiar consejos, información, y toda clase de valoraciones para ayudarse mutuamente respecto a las asignaturas en cuestión. Con el fin de encontrar rápidamente las asignaturas de preferencia de quien usa la aplicación, la herramienta proporciona un sistema de favoritos para añadir y quitar asignaturas de su lista de favoritos personal. Las tecnologías utilizadas para realizar la aplicación incluyen Django y Django Rest Framework para el backend, y React.js y Tailwind CSS para el frontend. Este sistema surge ante la necesidad de disponer de una plataforma que aúne en un mismo lugar información clara y accesible sobre las asignaturas de la universidad, así como valoraciones de otras personas usuarias. La falta de una aplicación con estas características motivó el desarrollo de esta herramienta, que busca fomentar una toma de decisiones más informada y colaborativa entre el alumnado.
